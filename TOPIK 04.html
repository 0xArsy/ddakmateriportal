Conceptual Deep Dive

Mental model & terminologi penting dalam logic gates

Mengapa Ada Gate Lain Selain AND/OR/NOT?

Set {AND, OR, NOT} sudah cukup untuk membangun semua fungsi Boolean (disebut complete set of logic). Namun, gate tambahan digunakan karena:

  • ▸Ekonomis & efisien — NAND/NOR alami di teknologi CMOS dari sisi luas chip dan kecepatan.
  • ▸Self-sufficient — NAND dan NOR masing-masing dapat merepresentasikan semua fungsi logika (universal gates).
  • ▸Fungsional spesifik — XOR sangat berguna untuk parity generation, adder, counter.
⚡ Klasifikasi Gate:
Primitive gate = menggunakan satu tipe operasi primitif (AND/OR) + opsional inversi. Contoh: NAND, NOR.
Complex gate = membutuhkan lebih dari satu tipe operasi. Contoh: XOR, XNOR.

Key Terms

Universal Gate

Tipe gate yang dapat mengimplementasikan semua fungsi Boolean sendirian. NAND dan NOR adalah universal gates.

SOP (Sum of Products)

Ekspresi Boolean sebagai jumlah (OR) dari beberapa product (AND). Contoh: F = AB + C'D + E

POS (Product of Sums)

Ekspresi Boolean sebagai product (AND) dari beberapa sum (OR). Contoh: G = (A+B)·(C'+D)·E

Don't Care

Kombinasi input yang tidak akan pernah terjadi, diberi simbol X pada K-map. Dapat digunakan untuk optimasi.

Common Pitfalls

⚠ NAND/NOR bukan asosiatif!

NAND dan NOR tidak memiliki simbol operasi standar karena operasinya tidak asosiatif. Jadi (A NAND B) NAND C ≠ A NAND (B NAND C). Gunakan ekspansi De Morgan untuk manipulasinya.

⚠ XOR/XNOR hanya untuk 2 input!

Symbol XOR/XNOR berbentuk hanya ada untuk 2 input. Untuk lebih dari 2 variabel, disebut odd function (XOR) atau even function (XNOR), bukan XOR.

⚠ Inverter dengan 1 input

NAND gate dengan satu input menjadi inverter (karena (x·x)' = x'). Begitu juga NOR dengan satu input.

De Morgan & Representasi Ganda

Setiap gate NAND/NOR memiliki dua representasi ekivalen berkat hukum De Morgan:

NAND (3-input)
AND-Invert: F = (X·Y·Z)'
Invert-OR: F = X' + Y' + Z'
NOR (3-input)
OR-Invert: F = (X+Y+Z)'
Invert-AND: F = X'·Y'·Z'
✓ Mental Model: Memiliki kedua representasi mempermudah visualisasi fungsi circuit. Pilih representasi yang paling intuitif untuk konteks yang ada.

NAND = Universal Gate

Bukti bahwa {NAND} adalah complete set: implementasikan NOT, AND, OR hanya dengan NAND.

NOT dengan NAND
(x·x)' = x'

Sambungkan input yang sama ke dua input NAND → hasilnya adalah inverter.

Prinsip: Idempotency
AND dengan NAND
((x·y)'·(x·y)')' = x·y

Output NAND(x,y) diinverskan lagi dengan NAND 1 input → AND.

Prinsip: Idempotency + Involution
OR dengan NAND
((x·x)'·(y·y)')' = x+y

Invert masing-masing input, lalu NAND hasilnya → De Morgan memberikan OR.

Prinsip: Idempotency + DeMorgan + Involution
NOR = Universal Gate

Bukti bahwa {NOR} adalah complete set: implementasikan NOT, AND, OR hanya dengan NOR.

NOT dengan NOR
(x+x)' = x'

Sambungkan input yang sama ke dua input NOR → hasilnya inverter.

Prinsip: Idempotency
OR dengan NOR
((x+y)'+(x+y)')' = x+y

Output NOR(x,y) diinverskan lagi dengan NOR 1 input → OR.

Prinsip: Idempotency + Involution
AND dengan NOR
((x+x)'+(y+y)')' = x·y

Invert masing-masing input, lalu NOR hasilnya → De Morgan memberikan AND.

Prinsip: Idempotency + DeMorgan + Involution
SOP → NAND Circuit

Ekspresi SOP dapat diimplementasikan langsung sebagai 2-level NAND circuit.

Contoh: F = A·B + C'·D + E

AND-OR → NAND-NAND

Caranya: ganti semua AND gate level-1 dan OR gate level-2 dengan NAND gate. Input tunggal (seperti E) harus diinversi terlebih dahulu jika ada.

✓ Insight: 2-level AND-OR = 2-level NAND-NAND secara fungsional. Bubbles (inversi) berpasangan dan saling menghilangkan.
POS → NOR Circuit

Ekspresi POS dapat diimplementasikan langsung sebagai 2-level NOR circuit.

Contoh: G = (A+B)·(C'+D)·E

OR-AND → NOR-NOR

Caranya: ganti semua OR gate level-1 dan AND gate level-2 dengan NOR gate. Input tunggal (seperti E) harus diinversi terlebih dahulu.

✓ Insight: 2-level OR-AND = 2-level NOR-NOR secara fungsional. Prinsipnya identik dengan SOP-NAND tetapi untuk POS.

Q1
NAND Gate

Manakah pernyataan yang BENAR tentang NAND gate dengan 3 input X, Y, Z?

METODOLOGI PENYELESAIAN

NAND = NOT AND. Untuk 3 input X, Y, Z:

F = (X·Y·Z)' [AND-Invert / symbol NAND]

Dengan hukum De Morgan: (X·Y·Z)' = X' + Y' + Z'

F = X' + Y' + Z' [Invert-OR / symbol alternatif]

Jawaban D mengacu pada representasi NOR (Invert-AND = X'·Y'·Z' adalah representasi NOR). Jawaban B adalah NOR gate. Jawaban A adalah OR. Hanya C yang benar karena menjelaskan NAND dengan kedua bentuknya sesuai De Morgan.

Q2
XOR Identitas

Jika A = 1, berapakah hasil dari A ⊕ 1?

METODOLOGI PENYELESAIAN

Identitas XOR: X ⊕ 1 = X' (invers dari X)

A = 1 → A ⊕ 1 = A' = 1' = 0

Ingat identitas XOR lainnya: X⊕0=X, X⊕X=0, X⊕X'=1. Jadi XOR dengan 1 selalu menginversi input, bukan memberikan nilai tetap.

Q3
Universal Gates

Mana dari pilihan berikut yang merupakan implementasi OR menggunakan hanya NAND gates?

METODOLOGI PENYELESAIAN

Langkah:

1. NAND(x, x) = (x·x)' = x' (idempotency)
2. NAND(y, y) = (y·y)' = y' (idempotency)
3. NAND(x', y') = (x'·y')' (De Morgan: = x+y)
((x·x)'·(y·y)')' = (x'·y')' = x+y ✓

Pilihan D: NAND(NAND(x,y), NAND(x,y)) = ((xy)') = x·y, bukan x+y. Itu adalah AND!

Q4
Full Adder

Pada Full Adder, jika X=1, Y=1, Z=1 (carry-in), berapakah nilai S (Sum) dan C (Carry-out)?

METODOLOGI PENYELESAIAN

Gunakan persamaan Full Adder:

S = X ⊕ Y ⊕ Z = 1 ⊕ 1 ⊕ 1
= (1⊕1) ⊕ 1 = 0 ⊕ 1 = 1
C = X·Y + (X⊕Y)·Z
= 1·1 + (1⊕1)·1
= 1 + 0·1 = 1 + 0 = 1

Verifikasi: 1+1+1 = 3 = 11₂ → S=1 (bit bawah), C=1 (bit atas). ✓

Opsi D salah karena S≠0. Jumlah tiga 1 = 3, bukan 2.
Q5
SOP ke NAND / Latihan DDAK

Ekspresi SOP F = C'D + AB'C + BD' diubah ke bentuk hanya NAND dan Invert. Manakah yang benar?

METODOLOGI PENYELESAIAN

Konversi SOP ke NAND-only menggunakan identitas: A+B+C = ((A)'·(B)'·(C)')' (De Morgan)

F = C'D + AB'C + BD'
= (C'D + AB'C + BD')
= ((C'D)' · (AB'C)' · (BD')')' ← De Morgan pada OR level
F = ((C'D)'·(AB'C)'·(BD')')' ✓

Setiap product term dinegatif dengan NAND, lalu hasilnya dinegatif lagi dengan NAND (implementasi OR). Ini adalah pola 2-level NAND-NAND yang ekivalen dengan AND-OR (SOP).

Pilihan D adalah konversi ke POS/NOR, bukan NAND.
