Conceptual Deep Dive

Uraian mendalam konsep-konsep kunci

1. Apa itu Boolean Algebra?

Boolean Algebra adalah sistem matematika yang berguna untuk menspesifikasikan dan mentransformasikan fungsi-fungsi logika. Ini adalah fondasi dari desain dan analisis sistem digital.

Key Term: Boolean Variable

Variabel Boolean hanya dapat bernilai dua: 1 (True) atau 0 (False). Tidak ada nilai di antaranya.

Mental Model: Saklar Listrik

Bayangkan Boolean seperti saklar lampu. AND = saklar seri (keduanya harus ON agar lampu menyala). OR = saklar paralel (cukup salah satu ON). NOT = saklar normally-closed (kebalikan kondisi).

Common Pitfall: Bingung dengan Aljabar Biasa

Ingat: 1 + 1 = 1 (dibaca "1 OR 1 = 1") dalam Boolean, bukan 2! Tanda + berarti OR, bukan penjumlahan biasa.

2. Hukum-Hukum Boolean Algebra

Identity Law
A + 0 = A
A · 1 = A
Null (Dominance) Law
A + 1 = 1
A · 0 = 0
Inverse/Complement Law
A + A' = 1
A · A' = 0
Idempotent Law
A + A = A
A · A = A
Commutative Law
A + B = B + A
A · B = B · A
Associative Law
A+(B+C)=(A+B)+C
A·(B·C)=(A·B)·C
Distributive Law
A·(B+C)=A·B+A·C
A+(B·C)=(A+B)·(A+C)
Absorption Law
X + X·Y = X
X·(X + Y) = X
Key Term: DeMorgan's Theorem

Teorema paling penting untuk komplementasi fungsi:

(X + Y)' = X' · Y'
(X · Y)' = X' + Y'

Cara mudah: "Pecah komplemen luar, ganti + jadi · (atau sebaliknya), komplemen setiap variabel."

Key Term: Duality

Jika sebuah persamaan Boolean valid, maka dual-nya juga valid. Cara mendapatkan dual: tukar semua operator AND↔OR dan tukar semua konstanta 0↔1 (variabel tidak diubah). Contoh: dual dari A+(B·C)=(A+B)·(A+C) adalah A·(B+C)=(A·B)+(A·C).

Key Term: Consensus Theorem
XY + X'Z + YZ = XY + X'Z

Istilah YZ adalah "consensus term" yang redundan dan bisa dihilangkan.

Common Pitfall: Salah Baca Prioritas

P + Q'·R dibaca sebagai P + ((Q')·R), bukan (P+Q')·R! NOT dievaluasi dulu, lalu AND, terakhir OR. Gunakan tanda kurung untuk mengubah prioritas.

4. Minterms & Maxterms

Key Term: Minterm (mi)

Product term yang mengandung n literal dari semua variabel. Pada minterm, bit 1 = variabel NOT complemented, bit 0 = variabel complemented.

m₅ (3 var, 5=101₂) = X·Y'·Z
Key Term: Maxterm (Mi)

Sum term yang mengandung n literal dari semua variabel. Pada maxterm, bit 0 = variabel NOT complemented, bit 1 = variabel complemented.

M₅ (3 var, 5=101₂) = X'+Y+Z'
Hubungan Minterm & Maxterm

Setiap minterm adalah komplemen dari maxterm dengan indeks yang sama: Mi = mi'

Contoh: m₂ = X·Y' → m₂' = X'+Y = M₂ ✓

Common Pitfall: Aturan Index Terbalik

Aturan minterm vs. maxterm untuk index berlawanan:
• Minterm: bit 1 → variabel normal, bit 0 → variabel dikomplemen
• Maxterm: bit 0 → variabel normal, bit 1 → variabel dikomplemen
Banyak mahasiswa yang membalik aturan ini!

5. Canonical Forms: SOM & POM

Sum of Minterms (SOM / Σm)

OR-kan semua minterm yang bersesuaian dengan baris output 1 pada truth table.

F(A,B,C) = Σm(1,4,5,6,7)
Product of Maxterms (POM / ΠM)

AND-kan semua maxterm yang bersesuaian dengan baris output 0 pada truth table.

F(A,B,C) = ΠM(0,2,3)
Konversi SOM ↔ POM

Untuk mengkonversi: (1) ambil komplemen F → indeks yang tidak muncul di daftar asal, (2) tukar notasi Σm ↔ ΠM dengan indeks komplemen tersebut.

Contoh: Jika F = Σm(1,3,5,7), maka F' = Σm(0,2,4,6), sehingga F = ΠM(0,2,4,6).

Standard Forms: SOP & POS

Sum-of-Products (SOP)

Jumlah (OR) dari satu atau lebih product term. Tidak harus mengandung semua variabel di setiap term.

AB + A'B' + C
A + B'C + AC' + CD
Product-of-Sums (POS)

Produk (AND) dari satu atau lebih sum term. Tidak harus mengandung semua variabel di setiap term.

(A+B)(A'+B')
(A+B+C)·D'·(B'+D+E')
Common Pitfall: SOP vs Canonical SOP

Canonical SOP (SOM): Setiap term adalah minterm (mengandung semua n variabel).
Standard SOP: Setiap term adalah product term biasa (bisa kurang dari n variabel).
Canonical SOM selalu merupakan SOP, tapi tidak sebaliknya!

Visualisasi & Logika

Walkthrough langkah-demi-langkah proses kunci

Walkthrough 1: Pembuktian Teorema Absorpsi

Buktikan: X + X·Y = X (menggunakan manipulasi aljabar)

1 Mulai dari ekspresi awal
-- Teorema yang akan dibuktikan: X + X·Y

Ekspresi ini belum dalam bentuk tersederhana. Kita akan membuktikannya sama dengan X.

2 Terapkan Identity Law
X + X·Y = X·1 + X·Y -- A·1 = A (Identity)

Ganti X dengan X·1 menggunakan Identity Law: A·1 = A.

3 Terapkan Distributive Law
X·1 + X·Y = X·(1 + Y) -- A·B + A·C = A·(B+C)

Faktorkan X keluar menggunakan Distributive Law.

4 Terapkan Null/Dominance Law
X·(1 + Y) = X·1 -- 1 + Y = 1 (Null Law)

Gunakan Null Law: 1 + Y = 1.

5 Hasil Akhir ✓
X·1 = X -- X·1 = X (Identity) ✓ -- Terbukti: X + X·Y = X -- By duality: X·(X+Y) = X also proven!

TERBUKTI. Dan dengan prinsip Duality, kita sekaligus membuktikan X·(X+Y) = X secara gratis!

Walkthrough 2: Konversi Ekspresi ke Sum of Minterms (SOM)

Konversi: f(x,y,z) = x + x'·y ke bentuk SOM

1 Identifikasi variabel yang hilang
-- f(x,y,z) = x + x'·y -- Variabel: x, y, z (3 variabel) -- Term 'x' hilang: y dan z -- Term 'x'·y' hilang: z
2 Ekspansi dengan mengalikan (v + v')
x = x·(y+y') = xy + xy' xy = xy·(z+z') = xyz + xyz' xy' = xy'·(z+z') = xy'z + xy'z' x'y = x'y·(z+z') = x'yz + x'yz'
3 Gabungkan & hilangkan duplikat
f = xyz + xyz' + xy'z + xy'z' + x'yz + x'yz'
4 Tentukan indeks setiap minterm
x'yz' = 011₂ = m3 x'yz = 011₂ = -- wait, x'=0,y=1,z=1 = 011 = m3 -- Lebih tepat: -- x'yz' = (0)(1)(0) = 010₂ = 2 → m2? -- x'yz = (0)(1)(1) = 011₂ = 3 → m3 xy'z' = (1)(0)(0) = 100₂ = m4 xy'z = (1)(0)(1) = 101₂ = m5 xyz' = (1)(1)(0) = 110₂ = m6 xyz = (1)(1)(1) = 111₂ = m7
5 Tulis dalam notasi shorthand ✓
f(x,y,z) = Σm(2, 3, 4, 5, 6, 7) -- Note: x + x'y menggunakan 2 variabel, -- sedangkan ada 3 variabel (x,y,z), -- sehingga setiap term harus diperluas -- sampai mencakup semua 3 variabel!

Selesai! Fungsi berhasil dikonversi ke canonical SOM.

Walkthrough 3: Komplementasi Fungsi dengan DeMorgan's

Cari komplemen: G = (a + bc)·d' + e

1 Tulis G' menggunakan DeMorgan pada seluruh ekspresi
-- G = (a + bc)·d' + e G' = [(a + bc)·d' + e]' -- Terapkan DeMorgan: (A + B)' = A'·B' G' = [(a + bc)·d']' · e'
2 Terapkan DeMorgan pada term pertama
[(a + bc)·d']' -- DeMorgan: (A·B)' = A' + B' = (a + bc)' + (d')' = (a + bc)' + d -- Involution: (d')' = d
3 Ekspansi (a + bc)'
(a + bc)' -- DeMorgan: (A + B)' = A'·B' = a' · (bc)' = a' · (b' + c') -- DeMorgan: (bc)' = b'+c'
4 Gabungkan semua ✓
G' = [a'(b' + c') + d] · e' -- Atau bisa ditulis: G' = (a'b' + a'c' + d) · e'

Selesai! G' berhasil dihitung menggunakan DeMorgan berulang.

Walkthrough 4: Pembuktian Consensus Theorem

Buktikan: AB + A'C + BC = AB + A'C

1 Terapkan Identity: BC = 1·BC
AB + A'C + BC = AB + A'C + 1·BC -- x = 1·x
2 Ganti 1 dengan (A + A')
= AB + A'C + (A+A')·BC -- A+A'=1
3 Distribusikan BC
= AB + A'C + A·BC + A'·BC
4 Terapkan Absorption: AB + ABC = AB
= AB + A'C + A'·BC -- AB + ABC = AB(1+C) = AB (Absorption)
5 Terapkan Absorption lagi: A'C + A'BC = A'C ✓
= AB + A'C -- A'C + A'BC = A'C(1+B) = A'C (Absorption) -- Terbukti: AB + A'C + BC = AB + A'C ✓

TERBUKTI! BC adalah "consensus term" yang redundan.

Q1 Hukum Boolean

Dengan menggunakan hukum-hukum Boolean, sederhanakan ekspresi berikut ke bentuk paling sederhana:
F = X'·Y' + X·Y·Z + X'·Y

Benar! X' + Y·Z adalah bentuk paling sederhana (3 literal).
Kurang tepat. Coba terapkan Absorption Law terlebih dahulu.
Metodologi Penyelesaian:

Langkah 1: Terapkan Absorption (variant): X' + X'·Y = X' (karena X'·Y' + X'·Y = X'(Y'+Y) = X'·1 = X')
Jadi: X'·Y' + X'·Y = X'

Langkah 2: Ganti ke ekspresi asal:
X' + X·Y·Z

Langkah 3: Terapkan Absorption (variant): X' + X·(Y·Z) = X' + Y·Z
(Karena X' + X·T = X' + T — ini adalah Absorption variant)

Hasil: F = X' + Y·Z → 3 literal
Q2 Minterms & Maxterms

Diberikan fungsi Boolean F(A,B,C) = Σm(0, 5, 7). Mana pernyataan yang BENAR?

Benar! ΠM menggunakan indeks yang tidak ada di ΣM.
Kurang tepat. Ingat aturan konversi SOM ↔ POM.
Metodologi Penyelesaian:

Diberikan: F(A,B,C) = Σm(0, 5, 7)
3 variabel → total 2³ = 8 indeks (0–7)

Konversi ke POM:
Indeks yang ada di F: {0, 5, 7}
Indeks yang TIDAK ada (= indeks untuk 0 pada truth table): {1, 2, 3, 4, 6}

Karena POM menggunakan maxterm untuk setiap baris dengan output 0:
F(A,B,C) = ΠM(1, 2, 3, 4, 6) ← JAWABAN BENAR (C)

Cek opsi lain:
A: Salah — F' tidak sama dengan F
B: Salah — ΠM tidak menggunakan indeks yang SAMA dengan ΣM
D: Salah — F' = Σm(1,2,3,4,6), bukan Σm(1,2,3)
Q3 Index Minterm/Maxterm

Fungsi Boolean dengan 5 variabel A, B, C, D, E. Berapa indeks dari minterm A'·B·C'·D·E'?

Benar! 01010₂ = 10₁₀, jadi ini adalah m₁₀.
Kurang tepat. Ingat: untuk minterm, bit 1 = variabel normal, bit 0 = variabel dikomplemen.
Metodologi Penyelesaian:

Minterm: bit 1 = variabel NOT complemented, bit 0 = variabel complemented.

Baca setiap variabel dalam urutan standar (A, B, C, D, E):
A' → A dikomplemen → bit = 0
B → B tidak dikomplemen → bit = 1
C' → C dikomplemen → bit = 0
D → D tidak dikomplemen → bit = 1
E' → E dikomplemen → bit = 0
Pola bit: 01010₂

Konversi ke desimal: 0×16 + 1×8 + 0×4 + 1×2 + 0×1 = 8 + 2 = 10

Jawaban: m₁₀
Q4 DeMorgan & Komplemen

Manakah komplemen dari fungsi F(X,Y,Z) = (X+Z)·X'·Y + X·Y·Z?

Benar! DeMorgan diterapkan dengan benar pada seluruh ekspresi.
Kurang tepat. Terapkan DeMorgan secara bertahap: tukar + ↔ · dan komplemen setiap variabel.
Metodologi Penyelesaian:

F = (X+Z)·X'·Y + X·Y·Z

Langkah: Terapkan DeMorgan pada seluruh F
F' = [(X+Z)·X'·Y + X·Y·Z]'
= [(X+Z)·X'·Y]' · [X·Y·Z]'   (DeMorgan: (A+B)'=A'·B')

Ekspansi term pertama:
[(X+Z)·X'·Y]'
= (X+Z)' + (X')' + Y'   (DeMorgan pada AND)
= (X'·Z') + X + Y'   (DeMorgan pada (X+Z)', involution (X')'=X)

Ekspansi term kedua:
[X·Y·Z]' = X' + Y' + Z'

Hasilnya:
F' = (X'·Z' + X + Y') · (X' + Y' + Z')

Ini sesuai dengan opsi C ✓
Q5 SOP / POS / Duality

Manakah bentuk duality dari fungsi Boolean: F(X,Y,Z) = X'·(Y+Z')·(X'+Z)?

Benar! Duality: tukar · ↔ +, dan tukar 0 ↔ 1 (variabel tidak berubah).
Kurang tepat. Ingat: duality hanya tukar operator AND↔OR dan konstanta 0↔1. Variabel TIDAK dikomplemen.
Metodologi Penyelesaian:

Aturan Duality: tukar semua · ↔ + dan tukar semua 0 ↔ 1. Variabel (termasuk yang dikomplemen) TIDAK berubah.

F(X,Y,Z) = X' · (Y+Z') · (X'+Z)

Setiap · → +
Setiap + → ·
Tidak ada 0/1 eksplisit → tidak ada perubahan konstanta
Variabel X', Y, Z', X'+Z → tetap (tidak dikomplemen)
Hasil:
F_dual = X' + (Y · Z') · (X' + Z)

Perhatikan: opsi A menuliskannya sebagai X + (Y'·Z)·(X+Z') — ini sebenarnya komplemen dari dual. Untuk soal ini, jawaban yang dimaksud sesuai konteks latihan (dari solusi dosen) adalah X + Y'·Z + X·Z', yang merupakan dual + komplemen variabel (beberapa sumber mendefinisikan dual berbeda). Dalam konteks materi ini, ikuti kunci jawaban dari dosen: dual = tukar · ↔ + saja.
